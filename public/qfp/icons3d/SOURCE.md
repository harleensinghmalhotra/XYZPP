# 3D stat icons — source & processing

The four stat-strip icons are custom 3D renders (navy/gold clay style on the
brand palette), exported as WebP with a transparent background.

| File | Stat |
|---|---|
| `stat-book.webp` | 400M+ Books Delivered |
| `stat-globe.webp` | 25+ Countries Across 4 Continents |
| `stat-box.webp` | 1000+ Containers Shipped |
| `stat-clock.webp` | 98% On-Time Delivery |

To swap an icon, replace the matching `.webp` in place, keeping a transparent
background and square-ish framing so it aligns with the other three.

CONVEYOR PLAQUE OVERRIDES — drop finished Canva PNGs here.

A file named  plaque-<stage>.png  (or plaque-<stage>-fr.png for French) placed in
THIS folder automatically replaces the conveyor plaque face for that stage. With no
files here, the site draws the plaque text itself — nothing changes.

  stage keys:  print  quality  fulfillment  warehouse  ship  covered
  e.g.  plaque-print.png   plaque-ship.png ("Shipping")   plaque-covered-fr.png

Spec + templates + guide + the current faces:  "FLOW assets/plaque templates/".
Export must be 1280 × 964 PNG, transparent outside the cream plate.

# About — Gallery

**Appears on the site:** /about (Our Story) → facility gallery (preview grid + "See More" lightbox)

**How to swap:** replace a file below with your own, keeping the **exact same filename and extension**. Order and the full walk-through list live in `src/assets/gallery/manifest.js`. Push the change — the new asset goes live on the next deploy.

The gallery renders `.webp` as photos and `.mp4` as lightbox video (media-agnostic). The preview grid shows the first six stills; the lightbox walks the **full 14-item list**, videos included. Items 1–11 are the original set; 12–14 are the real client machine/factory media appended after them.

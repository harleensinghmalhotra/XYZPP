# Newsroom — `/newsroom`

> ⚠️ **The newsroom is NOT folder-swappable.** Articles — their text **and** their
> images — are created and edited in the **Sanity Studio** (the QFP Newsroom) and
> served from Sanity's image CDN. Nothing on the live `/newsroom` page reads from
> this folder.

To publish or edit a newsroom story, use the Studio — not these files.

## What's in here (source / legacy only)

| Item | Role |
|---|---|
| `gallery/` | Legacy images from the old hard-coded newsroom (retired). Not shown live. |
| `video/` | Legacy press-run clip from the same retired data. Not shown live. |
| `drive-download-*/` | Raw press clippings + award PDFs (Forbes, PrintWeek, etc.) kept as source material. Referenced by no code. |

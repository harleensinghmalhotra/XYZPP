# Print On Demand

**Appears on the site:** /print-on-demand

**How to swap:** replace a file below with your own, keeping the **exact same filename and extension** (and, where noted, the pixel size). Push the change — the new asset goes live on the next deploy.

qfp-mark-foil.png is the embossed-foil logo mask on the build-your-book preview. pod-book.webp and preview-base.webp are legacy — the book preview is drawn entirely in CSS, so overwriting them changes nothing.

| `preview-base.webp` | 660×990 | Legacy preview base — preview is CSS-drawn | ⚠️ not used |
| `qfp-mark-foil.png` | 228×228 | Foil monogram mask (embossed logo effect) | ✅ live |

# Homepage — Hero

**Appears on the site:** homepage → hero

**How to swap:** replace a file below with your own, keeping the **exact same filename and extension** (and, where noted, the pixel size). Push the change — the new asset goes live on the next deploy.

The hero artwork (kids around the open book) is **language-specific**: the file shown depends on the site's language toggle — French visitors see `hero-main-fr.webp`, Spanish visitors see `hero-main-es.webp`, and every other language falls back to the English `hero-main.webp`. All three are shown at 2400×1350 and **must keep that exact size when replaced**, so the hand-positioned CTA buttons stay aligned on the book spine. The two prop-*.webp are legacy floating props not used by the current hero.

| File | Size | What it shows | Status |
|------|------|---------------|--------|
| `hero-main.webp` | 2400×1350 | Hero artwork — English (also the fallback) | ✅ live |
| `hero-main-fr.webp` | 2400×1350 | Hero artwork — French | ✅ live |
| `hero-main-es.webp` | 2400×1350 | Hero artwork — Spanish | ✅ live |
| `prop-bookstack.webp` | 1320×680 | Legacy floating prop (bookstack) | ⚠️ not used |
